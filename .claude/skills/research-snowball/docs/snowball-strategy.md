# Snowball 추적 전략 상세

## 1. 추적 깊이 제한

### 기본 규칙
- **최대 추적 깊이: 3단계** (기본값)
  - Depth 1: 원본 논문의 직접 참고문헌
  - Depth 2: Depth 1 논문의 참고문헌
  - Depth 3: Depth 2 논문의 참고문헌
- 연구 분야가 좁고 논문 수가 적은 경우, 사용자 요청에 따라 깊이 확장 가능

### 깊이별 우선순위
- **Depth 1**: 모든 참고문헌을 전수 조사한다 (빠짐 불가)
- **Depth 2**: 연구 주제와 직접 관련된 참고문헌만 추적한다
  - 관련성 판단 기준: 제목/초록에 핵심 키워드가 포함되어 있는가?
- **Depth 3**: 핵심 논문(인용수 상위 20%)의 참고문헌만 추적한다

### 깊이 확장 조건
다음 조건 중 하나라도 해당하면 깊이를 1단계 더 확장한다:
- Depth N에서 발견된 신규 논문 비율이 30% 이상일 때
- 사용자가 명시적으로 깊이 확장을 요청했을 때
- 핵심 논문이 Depth N에서 처음 발견되었을 때

## 2. 중복 판정 기준

### 1차 판정: DOI 기반 (최우선)
- DOI가 동일하면 무조건 중복이다.
- DOI 형식 정규화: `https://doi.org/10.xxxx/yyyy` 형태로 통일 후 비교한다.
- 대소문자를 무시한다.

### 2차 판정: 제목 기반 (DOI 없는 경우)
- 제목에서 특수문자, 관사(a, an, the)를 제거한다.
- 소문자로 통일 후 비교한다.
- 유사도 90% 이상이면 동일 논문으로 간주한다.
- 유사도 70~89%이면 저자명까지 확인하여 판정한다.

### 3차 판정: 저자+연도 기반 (제목도 불확실한 경우)
- 제1저자 성(last name) + 출판 연도가 동일하고, 제목 유사도 50% 이상이면 동일 논문 후보로 표시한다.
- 최종 판정은 수동 확인으로 보류한다 (`[중복 의심]` 태그).

## 3. 종료 조건

### 필수 종료 조건 (하나라도 충족 시 종료)
1. **미수집 논문 0개**: 현재 depth에서 새로 발견된 미수집 논문이 없다.
2. **최대 깊이 도달**: 설정된 최대 깊이(기본 3)에 도달했다.
3. **관련성 임계값 미달**: 현재 depth에서 발견된 논문 중 연구 주제와 관련된 것이 10% 미만이다.

### 종료 시 체크리스트
종료 전 반드시 다음을 확인한다:
- [ ] 모든 depth의 결과 파일이 작성되었는가?
- [ ] 각 depth 파일에 커버리지 보고가 포함되었는가?
- [ ] DOI 목록이 완전한가?
- [ ] 중복 의심 논문이 남아있는가? (있으면 표시)
- [ ] 총 수집 논문 수와 신규 발견 수가 기록되었는가?

## 4. Forward Snowball (선택적)

기본은 Backward Snowball(참고문헌 추적)이지만, 필요 시 Forward Snowball도 수행한다:
- **Forward Snowball**: 해당 논문을 인용한 최신 논문을 추적
- Semantic Scholar의 `citations` 또는 Google Scholar의 "Cited by" 기능 활용
- 핵심 논문(인용수 상위 10%)에 대해서만 수행
- 최신 2년 이내 논문 위주로 탐색

## 5. 진행 상황 기록

각 depth 완료 시 `.ralph-tui/progress.md`에 다음을 기록한다:
```
### Snowball Depth {N} 완료
- 시작 논문: X개
- 추출 참고문헌: Y개
- 미수집 발견: Z개
- 신규 수집: W개
- 중복 스킵: V개
- 소요 시간: 약 N분
```
