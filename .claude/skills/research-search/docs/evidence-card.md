# 증거 카드 (Evidence Card)

## 증거 카드 템플릿

발견한 **모든** 논문에 대해 아래 형식의 카드를 작성한다.

```markdown
### [논문 제목]
| 항목 | 내용 |
|------|------|
| 저자 | 저자1, 저자2, ... |
| 연도 | YYYY |
| 저널 | 저널/학회명 |
| DOI | https://doi.org/... |
| 인용수 | N (출처: Semantic Scholar / OpenAlex) |
| 방법론 | 실험 방법, 데이터셋, 분석 기법 등 |
| 핵심 발견 | 주요 결과 2-3문장 요약 |
| 한계점 | 저자가 언급한 한계 또는 평가자가 발견한 한계 |
| 신뢰도 | 높음 / 보통 / 낮음 |
| 전문 상태 | `[전문 확인 - 티어N/source]` / `[전문 확보 대기 - Tier 3 필요]` / `[전문 불가 - 모든 티어 실패]` |
| 관련 키워드 조합 | 이 논문을 발견한 키워드 조합 |
```

### 작성 규칙

- **모든 항목을 빠짐없이** 채운다
- 정보를 확인할 수 없는 항목은 `미확인`으로 표기한다
- **방법론**은 가능한 구체적으로 기술한다 (샘플 수, 실험 조건, 통계 기법 등)
- **핵심 발견**은 수치가 있으면 수치를 포함한다
- **한계점**은 논문 자체의 한계와 평가자가 판단한 한계를 구분한다

## 신뢰도 등급 기준

### 높음

아래 조건 중 **2개 이상** 충족:
- Impact Factor가 높은 저널에 게재 (Nature, Science, 분야 top 저널)
- 인용수가 해당 분야 평균 대비 높음
- 유명 연구 기관/연구실 소속 저자
- 대규모 샘플/데이터셋 사용
- 재현 가능한 방법론 명시
- 사전 등록(pre-registration) 수행

### 보통

- 일반 peer-reviewed 저널에 게재
- 인용수가 평균 수준
- 방법론이 타당하나 특별히 우수하지 않음
- 샘플 크기가 적절

### 낮음

아래 조건 중 **1개 이상** 해당:
- Predatory journal 의심 (Beall's List 참조)
- MDPI 의심 저널 (MDPI 전체가 predatory는 아니지만 일부 저널 주의)
- 극소 인용수 (출판 후 충분한 시간이 경과했음에도)
- 방법론 기술이 불충분하여 재현 불가
- 이해충돌(conflict of interest)이 명백
- 통계적 문제가 의심됨

### 낮음 판정 시 처리

- 신뢰도 낮은 논문도 **수집은 한다** (커버리지 완전성 원칙)
- `[LOW-CREDIBILITY]` 태그를 부착한다
- 결과가 중요하거나 치명적이면 `findings/low_credibility_important.md`에 별도 기록:
  - "이런 결과가 있었으나 출처 신뢰도가 낮아 추가 검증 필요"

## DOI 목록 섹션

모든 findings 파일 하단에 반드시 `## DOI 목록` 섹션을 포함한다.

형식:
```markdown
## DOI 목록

1. 저자 (연도). 제목. 저널. DOI: https://doi.org/10.xxxx/xxxxx
2. 저자 (연도). 제목. 저널. DOI: https://doi.org/10.xxxx/xxxxx
3. 저자 (연도). 제목. 저널. DOI: 미확인
4. 저자 (연도). 제목. 저널. DOI: https://doi.org/10.xxxx/xxxxx [DOI 미검증]
```

### DOI 상태 분류

- **확인됨**: WebFetch로 `https://doi.org/<DOI>` 접근하여 논문 페이지로 리다이렉트 확인
- **미확인**: DOI 자체를 찾을 수 없는 경우 (arXiv 전용 논문 등)
- **[DOI 미검증]**: DOI는 있으나 WebFetch로 검증하지 못한 경우
