# 통합 분석 프레임워크

## 1. 논문 간 관계 매핑

### 인용 네트워크 구축 방법

#### 직접 인용 관계
1. 각 논문의 참고문헌 목록에서 다른 수집 논문을 찾는다.
2. 방향성 있는 인용 관계를 기록한다: `논문A → 논문B` (A가 B를 인용)
3. 피인용 횟수(in-degree)를 집계하여 **허브 논문**을 식별한다.

#### 공동 인용 관계
- 두 논문이 동일한 제3의 논문에 의해 함께 인용되면 공동 인용 관계가 있다.
- 공동 인용 빈도가 높을수록 주제적 유사성이 높다.

#### 서지 결합 관계
- 두 논문이 동일한 참고문헌을 공유하면 서지 결합 관계가 있다.
- 공유 참고문헌 수가 많을수록 방법론적/주제적 유사성이 높다.

### 관계 매핑 출력 형식
```markdown
### 인용 네트워크 요약
- 허브 논문 (피인용 상위 5개):
  1. [제목] — 피인용 N회
  2. ...
- 고립된 논문 (인용/피인용 없음): N개
- 인용 클러스터: N개 식별
  - 클러스터 1: [주제] — 논문 N편
  - 클러스터 2: [주제] — 논문 N편
```

## 2. 연구 갭 식별 방법

### 체계적 갭 식별 절차

#### 1단계: 주제 매트릭스 작성
- 행(row): 연구 대상/변수
- 열(column): 연구 방법/접근법
- 셀: 해당 조합의 논문 수
- **빈 셀이 연구 갭**이다.

```markdown
| 대상\방법 | RCT | 코호트 | in vitro | 메타분석 | 리뷰 |
|-----------|-----|--------|----------|---------|------|
| 변수 A | 3편 | 1편 | 5편 | 0편 | 1편 |
| 변수 B | 0편 | 2편 | 0편 | 0편 | 0편 |
| 변수 C | 1편 | 0편 | 3편 | 1편 | 2편 |
```

#### 2단계: 시간 갭 분석
- 연도별 논문 수를 집계한다.
- 최근 3년간 논문이 없는 하위 주제를 식별한다.
- 한때 활발했으나 최근 중단된 연구 라인을 식별한다.

#### 3단계: 결과 불일치 분석
- 동일 주제에서 상충하는 결과를 보이는 논문 쌍을 찾는다.
- 불일치의 원인을 분석한다:
  - 방법론 차이?
  - 대상 집단 차이?
  - 측정 도구 차이?
  - 샘플 크기 차이?

#### 4단계: 미해결 질문 도출
- 논문들의 "향후 연구 방향(Future Work)" 섹션을 종합한다.
- 여러 논문에서 공통적으로 제안하는 연구 방향을 우선순위로 둔다.

### 갭 우선순위 매기기
| 우선순위 | 기준 |
|---------|------|
| 높음 | 여러 논문이 동일 갭을 지적 + 실용적 중요성 높음 |
| 보통 | 일부 논문이 갭을 지적 또는 매트릭스에서 발견 |
| 낮음 | 이론적 가능성은 있으나 실용적 중요성 불명 |

## 3. 감사(Audit) 기준

### 감사 체크리스트

#### 검색 완전성 확인
- [ ] 모든 주요 데이터베이스를 검색했는가? (WebSearch, OpenAlex, Semantic Scholar, arXiv, Google Scholar)
- [ ] 모든 키워드 변형을 사용했는가? (최소 5개 이상)
- [ ] 영어 외 다른 언어 논문도 검색했는가? (해당 시)
- [ ] Snowball 추적을 수행했는가?

#### 핵심 저자 확인
- [ ] 주요 저자 5인의 전체 출판 목록을 확인했는가?
- [ ] 해당 저자의 최신 논문이 포함되었는가?

#### 핵심 저널 확인
- [ ] 관련 논문이 가장 많이 게재된 저널 상위 3개를 식별했는가?
- [ ] 해당 저널에서 추가 검색을 수행했는가?

#### 최신성 확인
- [ ] 지난 1년 이내의 최신 논문이 포함되었는가?
- [ ] 프리프린트(preprint) 서버도 확인했는가?

#### 교차 검증
- [ ] 수집된 논문 수가 해당 분야의 예상 규모와 일치하는가?
- [ ] 리뷰 논문의 참고문헌과 수집 목록을 대조했는가?

### 감사 결과 등급
| 등급 | 기준 |
|------|------|
| 완전(Complete) | 추가 검색에서 신규 논문 발견 0개 |
| 거의 완전(Near-Complete) | 추가 발견 1~3개 (전체의 5% 미만) |
| 불완전(Incomplete) | 추가 발견 4개 이상 (전체의 5% 이상) |
| 심각한 누락(Critical Gap) | 핵심 논문 누락 발견 |

## 4. 핵심 저자 식별 방법

### 정량적 지표
1. **논문 수**: 수집된 목록 내 논문 편수
2. **총 인용수**: 해당 저자의 관련 논문 인용수 합계
3. **h-지수**: 전체 h-지수 (Semantic Scholar 또는 Google Scholar에서 확인)
4. **최신 활동**: 가장 최근 관련 논문의 출판 연도

### 정성적 지표
1. **영향력**: 해당 분야의 주요 리뷰 논문 저자인가?
2. **다양성**: 다양한 하위 주제에 걸쳐 연구하는가?
3. **협력 범위**: 다른 기관/국가의 연구자와 협력하는가?
4. **방법론 선도**: 새로운 방법론을 개발/도입했는가?

### 저자/랩 정보 형식
```markdown
### [저자명]
| 항목 | 내용 |
|------|------|
| 소속 | |
| 직위 | |
| 수집 내 논문 수 | N편 |
| 총 인용수 | N회 |
| 주요 연구 주제 | |
| 최신 논문 연도 | |
| 협력 기관 | |
| Google Scholar | URL |
| ORCID | URL |
```
